# Ficticious Database Generator

Este repositório reúne scripts e ferramentas projetados para gerar dados sintéticos robustos voltados para estudos de Ciência de Dados, Engenharia de Dados e Business Intelligence.

O objetivo principal não é apenas criar dados aleatórios, mas sim simular cenários de negócios reais, incorporando:

Lógica de Negócio: Consistência entre tabelas (ex: estoque vs. vendas).

Linha Temporal: Dados distribuídos coerentemente ao longo do tempo.

Imperfeições ("Dirty Data"): Inserção proposital de erros (nulos, duplicatas, formatação incorreta) para treinar limpeza de dados e ETL.
